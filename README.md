Questions for ABB Accelerator Grand Finale

This folder contains curated question sets to prepare for the ABB Accelerator Grand Finale. Each subfolder focuses on a different angle: situational scenarios, specific design/implementation choices, core ML concepts and algorithms, and general interview questions.

Subfolders:
- `situational/` — scenario-based questions to test system behaviour and incident handling.
- `specific/` — targeted questions about architecture, tech choices, and design trade-offs.
- `ml_concepts/` — required ML/time-series/graph concepts and algorithms to review.
- `general/` — general product, deployment, and team-readiness questions.

Use these to run mock Q&A sessions, handouts for the team, or slide-ready prompts.
